# Github Notes

## Getting Started

```sh
git clone https://github.com/felipecesr/github-notes.git
cd github-notes/
```

## Install dependencies

```sh
npm install
```

## Serve with hot reload at localhost:8080
```sh
npm start
```

## Run unit tests

```sh
npm test
```
